# TypeScript Library

A TypeScript library for Node.js.

## Requirements

- Node.js v22.16.0 or higher
- npm v10.9.2 or higher

## Installation

```bash
npm install
```

## Development

- Build the project:
  ```bash
  npm run build
  ```

- Run linting:
  ```bash
  npm run lint
  ```

- Format code:
  ```bash
  npm run format
  ```

## Usage

```typescript
import { greet } from 'kafka-common-22-16-0';

const message = greet('World');
console.log(message); // Output: Hello, World!
```
